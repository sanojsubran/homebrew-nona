# homebrew-apps

Homebrew tap for tools by [sanojsubran](https://github.com/sanojsubran).

## Setup

```sh
brew tap sanojsubran/apps
```

## Tools

| Formula | Description |
|---------|-------------|
| [nona](https://github.com/sanojsubran/nona) | A filename normalizing tool |

## Installation

```sh
# Install any tool directly (tap is added automatically)
brew install sanojsubran/apps/<formula>

# Examples
brew install sanojsubran/apps/nona
```

## Updating

```sh
brew update && brew upgrade
```
